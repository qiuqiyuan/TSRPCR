Trend Strenght Resistence Potential Confirmation Reasoning Trading System
  - for short-term speculative / trend-following trading.

投资理念：中线趋势是一种较难打破的趋势，因此在中线趋势中跟随趋势交易能获得极大的收益；在趋势末期如果
出现了急涨急跌，趋势改变带来的收益不仅巨大而且十分迅速。
在这两种市场环境下，从事期市短线交易（从开仓到平仓若干天到小于一个月）可以获得巨大收益。

跟随中线趋势时，有两种最好的买点：股价从盘整进入趋势时；价格短线调整到阻力线时。

股价从盘整进入趋势时，应当根据盘整之前的趋势和强弱设置合理的进入趋势的判定条件。

在趋势行进中，经过一段时间后，价格往往与重要阻力线（上升趋势为支撑线，下降趋势为压力线）形成乖离，而后
股价一定会调整到该重要阻力线，此时为最好的买点。不同的趋势强度有不同的阻力线。

在趋势末期，如果出现暴涨和频繁震荡的情况，表明此时市场投机性极强，多见于上升趋势。由于累计上升幅度过大，
一旦一致预期打破，容易出现急跌。急跌之后，即便有可能打破了中期上升趋势，此后仍然会延续上涨趋势一段时间。
因此这种情况是短线操作获利最快的时间，但是如果对买卖点为把握出错，甚至可能导致亏损。


开仓前要考虑以下因素：
趋势
强弱
压力支撑位（阻力位）
空间
确认

开仓时要有非常明确的止损设置！

止损时要在与仓位相同方向的阻力位处。例如，做多需要止损，在已经决定止损而非在重要压力位观望的前提下，必须
等待股价从当前价格上升到重要阻力位附近，不必恐惧下跌，然后尽快止损。不必恐惧下跌的理由是开仓经过了理性
思考，必然不会有严重的方向错误。
